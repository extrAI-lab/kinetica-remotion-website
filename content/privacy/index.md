**Last updated: May 2026**

## Who we are

Kinetica is an open-source library of animated components. This website ([kinetica.dev](https://kinetica.dev)) is where you can browse, preview, and install those components.

## What data we collect

**We do not collect any personal data.**

This website is a static site. There are no user accounts, no sign-up forms, no tracking pixels, and no advertising scripts.

## Third-party services

The site links to **GitHub** for source code and component repositories. When you follow those links, GitHub's own privacy policy applies.

The CLI tool (`npx create-kinetica`) runs locally on your machine and does not send any data to our servers. It scaffolds files in your local project only.

## Analytics

At the time of writing, this site does not use any analytics service. If analytics are introduced in the future, this policy will be updated before they are enabled.

## Cookies

This website does not set any cookies.

## Changes to this policy

If we make material changes, the "Last updated" date at the top of this page will be revised. We recommend checking this page periodically.
